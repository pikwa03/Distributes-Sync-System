# Distributed Synchronization System

Sistem sinkronisasi terdistribusi dengan Raft Consensus, Distributed Queue, dan Cache Coherence Protocol.

## Arsitektur

- **Distributed Lock Manager**: Raft Consensus Algorithm
- **Distributed Queue System**: Consistent Hashing
- **Distributed Cache**: MESI Protocol
